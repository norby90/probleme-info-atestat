## 🚀 Quick Start

### Clone the Repository
```bash
git clone https://github.com/norby90/probleme-info-atestat.git
cd probleme-info-atestat
```

### Compile a Program
```bash
g++ problem_name.cpp -o program
```

### Run the Program
```bash
./program
```

### Using an IDE
1. Open your preferred IDE (CodeBlocks, CLion, VS Code, etc.)
2. Load the project folder
3. Select a `.cpp` file and click **Run/Execute**
