📘 (Atestat) – Problem Collection

📌 Description
This repository contains a collection of programming problems solved in C/C++ that I used for my Informatics Certification Project (Atestat).
The problems cover fundamental algorithmic concepts studied in high school computer science, including:
Working with numbers
Bitwise operations
Arrays
Strings
Mathematical decompositions
Basic algorithms and logic
Each problem is implemented clearly and efficiently, following standard C/C++ practices.

🛠 Technologies Used
-C++
-Standard Library (iostream, cstring, etc.)
-Console-based input/output

📂 Repository Structure
Each file in the repository represents a separate problem and contains:
The problem statement (in comments)
The implemented solution
Example input/output (where applicable)

🎯 Purpose
The purpose of this repository is to:
Demonstrate my understanding of fundamental programming concepts
Showcase problem-solving skills
Serve as preparation material for the Informatics certification exam
Provide useful examples for students studying similar topics

🧠 Topics Covered
Some of the main topics included:
Binary decomposition (e.g., writing a number as a sum of powers of 2)
String processing (e.g., removing vowels from a word)
Loops and conditional statements
Character arrays
Bit manipulation

🚀 How to Run the Programs
Clone the repository:
git clone https://github.com/norby90/probleme-info-atestat.git
Open the project in your preferred IDE (CodeBlocks, CLion, VS Code, etc.)
Compile and run any .cpp file:
g++ file_name.cpp -o program
./program
